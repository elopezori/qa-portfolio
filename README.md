# \# QA Portfolio

# 

# Proyecto de portfolio orientado a QA Automation desarrollado con ASP.NET Core MVC, C#, xUnit y Playwright.

# 

# El objetivo de este proyecto es demostrar conocimientos de automatización de pruebas, control de versiones con Git y buenas prácticas de calidad de software.

# 

# \## Tecnologías utilizadas

# 

# \* C#

# \* .NET

# \* ASP.NET Core MVC

# \* xUnit

# \* Playwright

# \* Git

# \* GitHub

# 

# \## Funcionalidades

# 

# La aplicación muestra una lista de usuarios a través de una interfaz web desarrollada con ASP.NET Core MVC.

# 

# \### Usuarios incluidos

# 

# \* Juan

# \* Ana

# 

# \## Pruebas automatizadas

# 

# Actualmente el proyecto incluye pruebas automatizadas UI con Playwright y xUnit:

# 

# \* Verificación de carga de la página de usuarios.

# \* Verificación de los encabezados de la tabla.

# \* Verificación de la existencia de la tabla de usuarios.

# \* Verificación de la visualización de usuarios esperados.

#
